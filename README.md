This project presents a Machine Learning-based Fake News Detection system that classifies news articles as Fake or Real.
The system uses text preprocessing techniques and multiple classification models to analyze and evaluate news credibility.

Technologies Used:
Python – Core programming language
Google Colaboratory – Development environment
Scikit-learn – Machine learning model implementation
pandas – Data preprocessing and manipulation
NumPy – Numerical computations
GitHub – Repository hosting

Dataset sourced from Kaggle:
fake.csv
true.csv

Models Implemented:
Logistic Regression
Decision Tree
Random Forest
Gradient Boosting
The best-performing model was selected based on evaluation metrics.
